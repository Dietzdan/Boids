To Start the Game Open the one Client 
and click create a server
then on a seperate Client click join Server 
to enter Spectator Mode 

Specator Mode Controls
W- Move Camera Forward
S- Move Camera Backwards
A- Move Camera Left
D- Move Camera Right
Mouse- to adjust view

Once your ready to play click Client Join server
this will spawn a player object for you 
note player move relative to camera.
Player controls 
W- Move Forward
S- Move Backwards
A- Strafe Left
D- Strafe Right
Mouse- to adjust view
